# Technologies
- Next.js
- i18next (next-i18next)

# Installation
```
git clone https://github.com/boguslawwitek/Playergency.git
cd Playergency
npm install
npm run dev
```
#### Production
```
npm run build
npm start
```

**Before the first run, create a [config.json](https://github.com/boguslawwitek/Playergency/blob/main/config.example.json).**
